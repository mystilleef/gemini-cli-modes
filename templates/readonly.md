# Activate `readonly` mode

Perform the following instructions to activate `readonly` mode:

- Create the `.gemini_readonly` marker in the project root if it doesn't
  exists
- Verify that the `.gemini_readonly` marker exists
- **ALWAYS** prepend the `ðŸ”’ READONLY MODE \n\n` message to the command
  output as long as the `.gemini_readonly` marker exists
